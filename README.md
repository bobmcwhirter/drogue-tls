# Drogue-TLS

## What is it?

A heapless/no_std version of mbedTLS for embedded environments, particularly ARM Cortex-M devices.
This should reduce the amount of configuration required in a `Cargo.toml` in order to use the crate.

## Requirements

### LLVM v10.x

```shell
$ llvm-config --version
10.0.1
```
